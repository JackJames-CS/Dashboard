Personal AI Mission Control – System Overview
Purpose

Create a central dashboard that organizes and automates my life systems, including:

School

Work

Personal projects

Calendar and schedule

Tasks and deadlines

Notes and resources

Finances

AI tools and workflows

The goal is to open one dashboard and instantly understand what I need to do, instead of checking multiple platforms.

Core Concept

Mission Control acts as a hub that connects to the tools I already use and organizes their information.

Main inputs:

Google Calendar

Gmail

Obsidian

ChatGPT

Claude

Claude Code

Mission Control collects information from these sources and turns them into:

tasks

schedules

project tracking

reminders

AI planning suggestions

Main Life Categories

The dashboard is organized into three major systems.

1. School

Displays and manages:

upcoming assignments

class schedule

study blocks

important course links

university emails and deadlines

Automation ideas:

emails containing deadlines create tasks

assignments create calendar reminders

AI suggests study time before deadlines

2. Work

Tracks job-related information:

upcoming shifts

hours worked

estimated income

shift reminders

Possible automation:

work shifts from calendar

income tracking based on hours worked

3. Personal Projects

Used for managing side projects such as:

coding projects

experiments

freelance ideas

AI workflows

Each project includes:

overview

tasks

notes

progress tracking
